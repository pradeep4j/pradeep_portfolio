# Pradeep Portfolio Starter

This is an original, editable static portfolio starter built in:

- HTML
- CSS
- JavaScript

## Structure

- `index.html` — page structure and content
- `styles.css` — styling, layout, responsive behavior, animations
- `app.js` — mobile nav, reveal-on-scroll, typing effect

## How to use

1. Download and extract the folder.
2. Open `index.html` directly in a browser, or serve the folder with any static server.
3. Edit text, links, colors, and section content as needed.

## Notes

- Replace the placeholder contact links in the contact section.
- You can split project data into JSON later if you want a more dynamic version.
- Fonts are loaded from Google Fonts.
